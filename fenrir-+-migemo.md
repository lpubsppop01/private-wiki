# fenrir + migemo
```
fenrir はデスクトップ検索＆ランチャーツール。C/Migemo を使ったかな検索に対応している。
launchy や検索チャームより便利かなーと思い試用。

※途中で面倒になりやめた。

------------------------------------------------------------------------

インストールと設定。

1. fenrir075c.zip を展開。c:/opt/fenrir075c にした（以降、"インストール先"）。
2. C/Migemo の配布物から migemo.dll と dict ディレクトリをインストール先にコピー。
3. BREGEXP.DLL をインストール先にコピー。
4. ${インストール先}/data/fenrir.ini を編集して MigemoMode を 3 に変更。

fenrir をタスクトレイから終了すると fenrir.ini が上書きされてしまう？
あと Windows 8.1 環境だと入力ウィンドウの縦サイズが小さすぎて入力中のテキストが確認しづらい。
この辺りで中止。

Windows 7 環境でも試してみたが、入力ウィンドウの縦サイズが小さい問題が出た。
ググり方が悪いのか、言及しているページに当たらない。
```
