# メインマシンのハードウェア更新＋再セットアップ2016/05
```
2016/05/21-2
あれこれセットアップ。
* ローカルアカウントを作成。
  一度 Microsoft アカウントでログインするようにしたところ、
  ユーザー作業フォルダ名が自動で決まってしまった。
  Microsoft アカウントで常用するにしても一度ローカルアカウントを作ってから切り替える方がいい。
* 初回起動時にすでに無線 LAN クライアント（PLANEX の GW-300S KATANA）が動作しており、
  初回セットアップ中に接続先を聞いてきた。
* ビデオカードもとりあえずインボックスドライバで動いていた。
  ひとまず Update をなくなるまで当てる。
* この段階で HDD を接続した。
  ドライブレターはデータ用を D、光学ドライブを E、旧システム用をその後ろに。
* ホスト名など設定。
* PackageManagement の設定。

2016/05/21
win10 のクリーンインストールまで完了。
久々すぎるせいかいろいろ手間取ったが、それ以上に問題があったのが以下。
* FDD と ATAPI 接続の DVD ライターが接続不可。
* FDD が接続不可のため win7 DSP 版が厳密にはライセンス違反。
* SSD のうまい固定方法がなくマスキングテープで固定。
* ねじ類とかケースの付属物をどこへやったか忘れてしまった。
ライセンスについては win10 アップグレード後はそちらのライセンスが適用されるが、
win7 時点では問題のあるやり方になる。

2016/05/20
注文した品をすべて受け取ってきた（初期不良チェックのため受け取りを今日まで控えた）。
予定としては以下の手順で進めたい。
- 予め win10 のインストールメディアを作成しておく。
- M/B、ビデオカードを交換＋SSD 装着。既存の HDD は一旦すべて外す。
- win7 をインストールしてアクティベーション後に win10 をアップグレードインストール。
すでに現在無償アップグレード版の win10 を使用しているが、
変更後のハードウェア構成を覚えさせるため再度行う。
- win10 をクリーンインストール。
- 旧システム用とデータ用の HDD を再接続。古い１台はそのまま外しておく。
- セットアップ。
今回は win10 で追加された PackageManagement を使用したい。今後何度か再インストールするはずなので。
あと PSO2 についてはファイルをコピーして初回のダウンロード時間を
短縮できるはずなので要確認。

2016/05/13
最終的に GTX960 2G を選択して注文した。
あと SSD 用の変換ブラケットが売り切れたため同等の別製品にしたのと
シルバーグリスの残りが見つからなかったためそれも追加した。
http://www.amazon.co.jp/dp/B01E4V2W42
http://www.amazon.co.jp/dp/B00O1X37ES
残念ながら速達が使えなかったため来週半ばの到着となる。

2016/05/12
GTX960 と近い Radeon R9 380 だとメモリバス幅が 256bits になる。
現行の Radeon R9 シリーズは HDCP 2.2 は対応していないのが弱点として挙げられるが、
4K の映像コンテンツなんてまだまだ手を出さないので問題なし。
MSI R9 380 GAMING 4G
http://www.amazon.co.jp/dp/B00ZOXZM0Q
http://www.amazon.co.jp/dp/B00ZPOBI0M

よくあるゲーム用途では x8 と x16 のパフォーマンスは誤差程度しか違わないらしい。
Sapphire NITRO R9 380 4G
http://www.amazon.co.jp/dp/B017D7HY6A
性能と静音性の両面でよい評価。ただ３万円を超える。

ここまでくると GTX960 2G で２万ちょっとのものを選んだほうがいい気がするが、
３年くらい使うことを考えるなら遠慮なく R9 380 でいい気もする。
どのみち 4K で満足に動くわけではないはずなんだよなー。
PSO2 界隈で言われるインスぺというのは NVIDIA Inspector という非公式ツールのこと。
これに相当する Radeon 用ツールというと RadeonPro というのがあるそうだが、
NVIDIA Inspector のようにはいかない模様。まだ迷う…。
んーーーーー。PSO2 を第一目的とするなら GTX 960 で決まりか。
```
