# Visual Studio 2013 での git 利用
```
VS2013 標準の git クライアント機能で困らない程度に動作するのだが、
origin の url スキームが https である必要がある。
Visual Studio の外（コマンドラインから git とか Git GUI とか）で clone したときはまりやすい。

素直に https で Visual Studio から複製すれば OK。
```
