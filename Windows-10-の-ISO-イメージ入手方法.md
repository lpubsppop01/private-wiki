# Windows 10 の ISO イメージ入手方法
```
2016/05/21
配布サイト:
https://www.microsoft.com/ja-jp/software-download/windows10https://www.microsoft.com/ja-jp/software-download/windows10

win7-8.1 でアクセスすると「今すぐアップグレード」「ツールを今すぐダウンロード」の２択。
それ以外（UA を iPhone などにすれば OK）でアクセスすると ISO イメージを直に DL できる。
---
Filename: Win10_1511_Japanese_x64.iso
MD5: A1D0E6F15C0C0111A78D3D8A7FE6318F
SHA-1: C1B7F88772907D9CD64B9540EA7AB0CCB5EE2326
---
ハッシュ値がググってヒットしたものとは異なるが、２回 DL して同値だったので
今はこちらが正解と思う。
```
