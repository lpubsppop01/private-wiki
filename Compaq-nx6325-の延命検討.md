# Compaq nx6325 の延命検討
```
父に譲渡した Compaq nx6325 の延命について。

http://h50146.www5.hp.com/products/portables/old/business/nx6325/

http://h50146.www5.hp.com/support/ordercenter/parts/spn_nx6325.html

以前は調子が悪いということで検討したが、
今回は Windows XP のサポートが来年で切れることから。
安いマシンを買い足した方が楽なのだが、受け取らせるのが一苦労と思われる。

今は手に入らないものもあるかもしれない。
※以下は以前調べたときの内容なので現状では違うかもしれない。

* Turion X64 TL-58 は 5800 円くらいで手に入るようだ。
  店舗での値段なのでオークションならもう少し安いかも。
  値段に数千円の差があるのと熱暴走が懸念されるので、TL-58でいいだろう。
  ※TL-60 までは定格？よく調べた方がいい。

* SSDは…
  SATA のバージョンで速度頭打ちになるので、
  安値のもので十分かを確認する必要がある。後は動作報告があるもの。
  安値なら 5000 円前後。
  ※OS と同時期で十分。

* メモリ。PC2-5300 以上の SODIMM を 2GB×2 が最大。
  2000 円× 2 くらいかな。

全て行うと 15000 円くらいか。

------------------------------------------------------------------------

さらに win7/8 のライセンスが必要。
win7 が無難だが、今回は軽さも重要なので迷う。

以下によると Turion x64 なら win7 がストレスなく動くらしい:
http://detail.chiebukuro.yahoo.co.jp/qa/question_detail/q1268500072
http://detail.chiebukuro.yahoo.co.jp/qa/question_detail/q1347393338

win8 のシステム要件に画面解像度 1366x768 というのがあるが、
これは XGA に緩和されているらしい。
インストールでこけなければ動きはこだわらなくてよいだろう。

Windows Blue が発売されたら状況が変わるかもしれないが、
win7 にこだわらないなら急ぐことはない。

------------------------------------------------------------------------

CPU とメモリをまず交換し、時期を見て SSD 換装と OS アップグレードという手順で
進めよう。

------------------------------------------------------------------------

ひとまずメモリだけ交換する（注文した）。
MSSE が派手にメモリを使うらしいので、そこだけでも対策すれば
ストレスは減るだろう。

------------------------------------------------------------------------
```
